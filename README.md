# 🛒 Smart Shopping List (Python)

Uma aplicação de console simples e eficiente desenvolvida em Python para gerenciamento de listas de compras. O projeto foca na interação dinâmica com o usuário e na organização de dados em listas.

## 🚀 Funcionalidades

- **Entrada Dinâmica:** O usuário define a quantidade de itens que deseja cadastrar.
- **Tratamento de Erros:** Validação para garantir que apenas números inteiros sejam aceitos na definição da quantidade.
- **Validação de Conteúdo:** Impede a inserção de itens vazios na lista.
- **Exibição Formatada:** Apresenta a lista completa (formato bruto) e uma visualização individualizada e numerada.

## 🛠️ Tecnologias Utilizadas

* **Python 3.x**
* Biblioteca Standard (nenhuma dependência externa necessária)

## 📖 Como Funciona?

O script utiliza estruturas fundamentais de programação:
1.  **Try/Except:** Para capturar erros de digitação do usuário.
2.  **While/For Loops:** Para controlar o fluxo de inserção de dados.
3.  **Functions:** Modularização do código para separar a lógica de entrada da lógica de exibição.
